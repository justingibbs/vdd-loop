## Which license applies to what

| Path / file | License |
| --- | --- |
| `/skills/**` | Apache-2.0 |
| `/templates/**` | Apache-2.0 |
| `/examples/**` | Apache-2.0 |
| `/docs/**` | CC BY 4.0 |
| `README.md`, `METHODOLOGY.md`, `PROJECT_BRIEF.md` | CC BY 4.0 |
| `evaluation-guidelines.md` | CC BY 4.0 |
| `gherkin-guidelines.md` | MIT (see Third-party material) |

**Rule of thumb:** if a file is meant to be *executed, copied, or scaffolded
into another project* (a template, a skill, an example artifact), it is
Apache-2.0. If a file is meant to be *read* (the methodology, the docs, the
guidelines prose), it is CC BY 4.0.

The full license texts live in:

- [`LICENSE`](./LICENSE) — Apache License 2.0
- [`LICENSE-docs`](./LICENSE-docs) — CC BY 4.0
